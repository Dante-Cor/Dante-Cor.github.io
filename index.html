Mi Trayectoria

Desarrollador Full Stack en Java con formación en Geomática, y experiencia en el análisis, procesamiento y gestión de datos geoespaciales. Competente en el uso de QGIS, Python y PostGIS para el análisis y visualización espacial, así como en el desarrollo backend con Java, Spring Boot y APIs REST para la gestión eficiente de datos. Apasionado por integrar tecnologías emergentes en soluciones geomáticas, combinando pensamiento analítico y buenas prácticas de desarrollo (Git, Docker, CI/CD, SCRUM) para crear soluciones geomáticas precisas, escalables y orientadas al impacto social.
